금연 도우미 모바일 웹 앱을 React + Tailwind CSS로 만들어줘. 
코르도바 기반으로 네이티브 앱처럼 작동해야 해.

주요 기능:
- 온보딩 페이지 (금연 시작일, 하루 흡연량, 담배 가격 입력)
- 24단계 레벨 시스템 (1시간부터 10년까지)
- 실시간 금연 시간 추적 및 표시
- 절약 금액 자동 계산
- SOS 긴급 도움 기능
- 건강 개선 정보 표시

디자인 요구사항:
- iOS 스타일의 모던한 UI
- 글래스모피즘 효과 적용
- 다크모드 지원
- 부드러운 애니메이션
- 모바일 최적화 (360-430px 너비)

LocalStorage를 사용해서 모든 데이터를 저장하고, 
페이지를 새로고침해도 데이터가 유지되도록 해줘.

🎨 디자인 세부화 프롬프트 (2단계)
디자인을 더 개선해줘:

색상 시스템:
- 메인 그라데이션: 보라색(#6366f1) → 분홍색(#ec4899)
- 성공 색상: 초록색 계열
- 경고 색상: 주황색 계열
- 배경: 다크 그라데이션 (검정 → 진한 남색)

글래스모피즘:
- backdrop-blur 효과
- 반투명 배경 (bg-white/10)
- 미묘한 테두리 (border-white/20)
- 그림자 효과

애니메이션:
- 페이지 전환: 부드러운 fade-in
- 카드 호버: scale 효과
- 버튼 클릭: 살짝 눌리는 효과
- 프로그레스 바: 자연스러운 증가 애니메이션
⚙️ 기능 추가 프롬프트 (3단계)
다음 기능들을 추가해줘:

1. 레벨 시스템 상세화:
   - 1시간, 8시간, 24시간, 3일, 1주일, 2주일, 1개월... 10년까지 24단계
   - 각 레벨마다 달성 시 축하 애니메이션
   - 프로그레스 바로 다음 레벨까지 진행률 표시

2. 통계 대시보드:
   - 금연 시간 (일/시간/분 형식)
   - 피우지 않은 담배 개수
   - 절약한 금액 (원화 표시)
   - 건강 회복 단계별 정보

3. SOS 기능:
   - 긴급 도움 버튼
   - 금연 동기부여 메시지 랜덤 표시
   - 심호흡 가이드 타이머
   - 대체 활동 제안

4. 설정 페이지:
   - 금연 시작일 수정
   - 담배 가격 수정
   - 하루 흡연량 수정
   - 데이터 초기화 기능

🎯 최적화 프롬프트 (4단계)
앱을 최적화해줘:

성능:
- React.memo로 불필요한 리렌더링 방지
- useCallback/useMemo 적절히 사용
- 애니메이션 GPU 가속 (transform, opacity만 사용)

사용자 경험:
- 로딩 상태 표시
- 에러 처리 및 사용자 친화적 메시지
- 햅틱 피드백 준비 (코르도바 플러그인 연동 가능하도록)
- 오프라인에서도 작동

모바일 최적화:
- 터치 제스처 최적화
- 안전 영역(safe area) 고려
- 가로/세로 모드 대응
- 소프트 키보드 대응
